HUTKO payment gateway integration for Paid memberships Pro.

== Description ==
[This Paid memberships Pro Hutko payment plugin](https://hutko.org/en/cms/wordpress/wordpress-paid-membership-pro/?utm_source=wordpress.org&utm_medium=cms) adds HUTKO as payment gateway to plugin Paid memberships Pro. HUTKO can accept VISA and MasterCard securely, quickly and easily on your store in minutes. Recurring payments, simple, straightforward pricing, top notch fraud intelligence, and 24/7 support.

= Start with HUTKO =
[Create Free HUTKO Account](https://portal.hutko.org/#/account/create)

== Features ==
* Customize your checkout window
* Easy configuration - only Merchant ID and Secret Key!

== Allowed currencies ==
With us your customers can make purchases in many currencies. HUTKO support  EUR, USD, GBP, RUB, UAH and some other.

== Tariffs ==
Only commission with payment. It is possible to adjust payment of the commission on itself, or to impose on the client.

[See current tariffs on the HUTKO website](https://hutko.org/en/tariffs-hutko/)

== Installation ==
1. Ensure you have latest version of plugin installed (Paid memberships Pro)
2. Upload the plugin files to the `/wp-content/plugins/`(or upload the plugin zip file using wp-admin.)
3. Activate the plugin through the 'Plugins' menu.
